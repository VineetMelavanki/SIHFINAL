## HerbTrace - Blockchain Herbal Traceability System

HerbTrace is a blockchain-based botanical traceability system for Ayurvedic herbs, providing end-to-end transparency from collection to consumer. This repository contains a proof-of-concept frontend that demonstrates scanning, tracing, and stakeholder portals for collectors, manufacturers, and consumers.

# HerbTrace - Blockchain Herbal Traceability System

HerbTrace is a blockchain-based botanical traceability system for Ayurvedic herbs, providing end-to-end transparency from collection to consumer. ## ðŸŒ¿ Project Overview

Problem: Lack of verifiable, tamper-resistant traceability in the Ayurvedic herb supply chain.

Solution: HerbTrace uses an immutable ledger (simulated in the demo) and QR-based consumer-facing verification to improve authenticity and sustainability.

## Key Features

- End-to-end herb traceability
- QR-based consumer verification
- Geo-tagged collection events
- Modular frontend for collectors, manufacturers, consumers, and admins
